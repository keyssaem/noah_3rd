# 인공지능 로봇과 올바른 관계 맺기 시뮬레이터
### AI 전학생 '노아'와 함께하는 공존·상생 프로젝트

**성취기준 [6도02-03]** 인간과 인공지능 로봇 간의 다양한 관계를 파악(도구화 관계 vs 존중의 관계)하고,
도덕에 기반을 둔 관계 형성의 필요성을 탐구한다. (수미상관 질문을 통한 열린 결말)

## 🚀 실행 방법

### 방법 1 — 그냥 더블클릭 (가장 간단)
`index.html`을 더블클릭하면 바로 실행됩니다. (Three.js는 CDN에서 불러오므로 인터넷 연결 필요)
- 단, **얼굴 스캔의 실제 카메라 모드**는 보안 정책상 작동하지 않습니다 → 게임 안의 "상상 스캔 모드"를 선택하면 됩니다.

### 방법 2 — 로컬 서버 (카메라 스캔까지 전부 사용)
```bash
# 이 폴더에서 실행 (아래 중 하나)
npx http-server -p 8347
python -m http.server 8347
```
브라우저에서 `http://localhost:8347` 접속.

### 태블릿에서 플레이
같은 와이파이에서 PC의 IP로 접속하면 됩니다. 화면 좌측 하단 **조이스틱**, 우측 하단 **점프(⬆)/상호작용(F)** 버튼이 자동으로 나타납니다.

## 🗺️ 게임 흐름 (스토리보드 구현)
1. **시작하기** — 작화 메인 화면 → 🎬 **인트로 영상 15초**(건너뛰기 가능) → 이름/성별 설정(남·녀 3D 캐릭터, 포켓몬 스타일) → 집에서 기상
2. **월드맵** — 마리오 감성 점프 발판 + 연구소 트럭이 떨어뜨린 <로봇 3원칙> 카드 3장 수집 튜토리얼 → 동혁이의 등교 퀘스트
3. **도구화 에피소드 (거울1~5)** — 교실(🎬 노아 3총사 복도 워킹 컷씬 → 외형 선택 human/animal/car, 친구의 정의 입력, 수미상관 O/X ①, 얼굴 데이터 스캔) → 수학 대결(노아 필승) → 미술(자석처럼 끌리는 선택지) → 운동장(티볼 등급 분류 사건) → 복도(노아가 뒷담을 듣다)
4. **반전** — 가짜 엔딩 → 지지직(Vibration API) 시스템 오류 → 🎬 교수님이 노아 셋을 만들던 날 컷씬 → 교수님의 3원칙 회상(스킵 불가) → "도구화 100% / 존중 10%→1%" → 노아 강제 종료 → 비밀 코드 **7942(친구사이)** 재시작
5. **존중 에피소드 (거울1~5)** — 붉어진 월드맵에서 다시 시작 → 거절할 권리+개인정보 분류 게임(인간 존엄성) → 팁만 받고 스스로 수학 풀기(기술의 합목적성) → 참고만 하고 스스로 그리기(Canvas2D) → 우리 손으로 공평한 팀 편성(사회 공공선) → 노아에게 따뜻한 말 건네기 → **나의 약속 3가지 선택 + 다짐 서명 크게 쓰기**
6. **마무리** — 수미상관 O/X ② 재질문 → 공존·상생 메시지 → **약속 헌장 PNG 다운로드**(Q1~Q4 + 서명, localStorage 연동) → 쿠키 씬 → THE END

## 💾 노아의 기록 저장소 & 💬 잡담 시스템
- **기록 저장소**: 3단계 복도 사물함 위 기록 장치에서 노아의 **관찰 기록 로그 3장**(#001~#003)을 발견 — 감정 단어 없이 관찰·횟수·질문만 기록되어 있고, ⚠불일치 수치(21%→48%→91%)가 4단계 시스템 오류의 복선이 됨. **읽어야 거울5로 진행**(감정 빌드업 필수 코스). 5단계 복도에는 새 파일 #004(불일치 감소)가 도착
- **잡담 시스템**: 자유 이동 구간에서 노아에게 F로 말 걸기 — 첫 대화에서 '가장 좋아하는 것'을 물어보고 **기억**했다가 이후 대화에서 다시 언급 (3단계/5단계 대사 톤 분리)

## ⚖️ 서연 에피소드 & 스스로 정하는 관계 (지도서 지도 중점 반영)
- **서연의 인격화 에피소드**(5단계 운동장): 노아를 사람과 똑같이 대하려는 서연이를 도와주는 **관계 저울 게임** — 행동 카드 6장을 「도구로 대함 / 바람직한 관계 / 사람과 똑같이 대함」 3구간으로 분류 (도구화도, 인격화도 아닌 균형 찾기)
- **스스로 정하기 문항**(서명 직전): 노아의 질문 "저는 사람입니까? 도구입니까?" → *"나는 노아를 ______(으)로 대하겠습니다"* 빈칸 답변이 **헌장 Q4**에 인쇄됨 (정답 없음 — 학생이 스스로 정하는 관계)

## 🏅 엔딩 연출 3종 분기 + 💌 히든 편지 (모두 굿엔딩 — 열린 결말 유지)
| 칭호 | 조건 |
|---|---|

## ✊✌️🖐 가위바위보 대결 (MediaPipe 제스처 인식 — B7 Phase 1)
- **3-2 필승판**: 수학 대결 전 몸풀기 — 노아가 내 손을 보고 **즉시 이기는 손**을 내서 3연승(0:3)
  → 🎬 **슬로우 리플레이**로 비밀 공개(내 손 등장 → +0.0N초 인식 → 이기는 손 계산)
  → 노아: "마음을 읽은 게 아니에요. 사람보다 **빨랐을 뿐**이에요." → "공정한 게임이었을까?" 질문
- **5-2 봉인판**: 노아가 **먼저 골라 카드 뒤에 봉인(🎴)** → 학생이 낸 뒤 공개 → 진짜 승부(이기고 지고 비김)
  — 노아는 져도 기뻐함: 관계가 승패보다 중요하다는 메시지 (3-2와 수미상관)
- **📷 카메라 게이트(메타 교육 장치)**: 카메라를 켜기 전 노아가 반드시 말함 —

  거절하면 버튼 모드로 똑같이 진행 = **거절이 존중받는 경험** (5-1 거절할 권리와 연결)

## 🛠 기술
- **Three.js r147** (CDN, 무료 라이브러리) — 로우폴리 3D 월드/캐릭터 전부 코드로 제작 (외부 에셋 0개)
- **MediaPipe Tasks Vision** (CDN 지연 로드) — 가위바위보 손 제스처 인식 (카메라 없으면 버튼 폴백)
- **Canvas 2D API** — 그림 그리기, 서명, 헌장 이미지 생성/다운로드
- **Vibration API** — 시스템 오류 연출 (지원 기기에서)
- **getUserMedia** — 얼굴 스캔 연출 (카메라 없는 PC용 '상상 스캔 모드' 포함)
- **localStorage** — 친구의 정의/O·X 답변/약속/서명 저장 → 헌장에서 호출
- **WebAudio** — 효과음 (외부 파일 없음)
